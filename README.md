# 🧗 꿀멩이

## 📌 프로젝트 소개

**꿀멩이**는 클라이밍 입문자 및 애호가를 위한 통합 플랫폼입니다.  
영상 기반 학습 콘텐츠를 등록 및 공유할 수 있으며, 댓글, 좋아요, 팔로우 등 **커뮤니티 기능**을 통해 사용자 간 소통을 지원합니다.  
또한 개인의 **운동 계획을 캘린더로 관리**하고, 외부 데이터를 **크롤링하여 난이도에 따른 운동 지점을 추천**하는 기능도 제공합니다.
